Description:

I created this game because I LOVE Brawlstars. I thought it would be cool if I could recreate it as a 2 player game. This way Computer players can experience a fun, basic version of the game. 2 player games with one player using WASD and the other using Arrows is always a blast, so why not implement it into this. Controller mode makes the game more skilled and more fun.

I wanted this games mechanics to be as accurate to the game as possible while keeping it simple. I tried to choose Iconic Brawlers, as well as balancing it between the classes. I chose Doug over Poco for the healing class because Poco would get demolished in this version, whereas Doug wouldnt.

One Challange I faced was the amount of errors and switching the code from AI to GitHub.
